src: source codes
bin: binary codes
lengthstat: the statistics (PDF and CDF) of job length
cost: contains the cost information files to used in simulation
jobEventDir: useless
memstat: memory statistics
simfailureTrace: contains the job trace sample files
vmhosts: the list of VM instances (to be loaded at the beginning of simulation)
prop.config: the key configuration file. This file is loaded by JobEmulator.java (the entry point)
reference: two key papers which are very helpful to read
